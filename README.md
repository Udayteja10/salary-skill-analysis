# salary-skill-analysis
# What Determines Data Science Salaries?

## Overview

This project analyzes global data science job salaries to identify the key factors influencing compensation, including experience level, job role, location, and remote work.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib

## Analysis Performed

* Salary vs Job Role
* Salary vs Experience Level
* Salary vs Remote Work
* Salary vs Country
* Combined Analysis: Role vs Experience

## Key Findings

* Experience level has the strongest impact on salary
* Executive roles earn significantly more than entry-level roles
* Location, especially US-based jobs, strongly influences salary
* Remote work does not consistently lead to higher pay
* Experience progression has a greater effect than job title

## Conclusion

Experience and location are the most important factors influencing data science salaries, while job role and remote work have a secondary impact.
